実行方法(実行ファイルの動かし方)

    実行するためのコマンドは以下の通りである。

    Java version

    java -jar primeministers.jar

    Python version

    make test

    MAC OX Xの場合には、アプリケーションForest.appをダブルクリックする方が簡便である。

    PrimeMinisters.app

    また、openコマンドでPrimeMinisters.appを開いても実行可能である。

    $ open PrimeMinisters.app
